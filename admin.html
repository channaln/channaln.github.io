Admin sign-in

This page is not linked from the public site. Set your own ADMIN_HASH in js/admin.js (see ADMIN.md). Use a long, unique password — client-side auth is not a substitute for edge protection on sensitive deployments.

Content dashboard

Posts are stored in this browser’s localStorage. To publish for everyone, use Export HTML and add the file under blog/ in your GitHub repo, then run Export JSON as a backup. Site font and color theme apply to the whole public site (theme updates immediately).

Posts

    Topics: letters and numbers only (commas between). Underscores and symbols are stripped on save.

    Publish to GitHub

    Create a classic Personal Access Token at GitHub → Settings → Developer settings → Personal access tokens with repo scope. Stored in this browser only — never committed.

    Edit post

    Paste images into the body area (cursor in the box) or drag a file onto it.
